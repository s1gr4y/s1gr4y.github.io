My Projects

Preview (Link embedded in image) About
pl4tform: A simple 3D platforming game made in C and OpenGL. It includes collision detection and response with basic movement physics.
JserveMe: A web server that can provide users files. Users connect to the site to download and upload files. Similar to cloud storage. It was made in Java 8+. I implemented HTTP request header and body parsing, JSON send response formating, and n-ary tree to represent file system.
The_Wall: A basic (Bulletin Board System) BBS-like server. Users connect to the server using their browser of choice and can chat anonymously (not securely). It was built with NodeJS and the ExpressJS framework.
Polyhedral Splines Modifier: A several month-long, WIP, project developing a native object modifier for generating polyhedral splines in the open-source 3D graphics software, Blender.
